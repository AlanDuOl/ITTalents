Welcome to IT Talents

To use this web app you have to register a new user or use the Admin User and log-in into the app.

This app is composed of two sections: Admin and Profiles. The sections can be accessed through the menu bar in the top using the corresponding dropdown menus.

The dropdown menu for each section is only available after you successfully log-in into the app.

Admin section

The admin section is used by the Admin User. It allows you to see all registered profiles.

The Admin User is not allowed to access the Profiles section.

Profiles section

The Profiles section is used by a Registered User and allows you to create, update, view and remove a profile.

The Registered User cannot access the Admin section.

To access the Profiles section, register a new user with the Register option in the navigation menu in the top of the page and then log-in into the app using the Login option in the same menu.
